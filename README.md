## 🇺🇸 English Version

A Full Stack Progressive Web App (PWA) developed to digitalize attendance tracking and create a token economy focused on students' financial education.

### 🎯 The Problem
Manual tracking of attendance and physical rewards created operational bottlenecks. There was no centralized system to analyze performance or automate the delivery of incentives.

### 💡 The Solution
Built a responsive web application acting as an "educational fintech." The system manages a virtual currency (Talents), allowing teachers to reward engagement. Students use a QR Code Badge to check-in and purchase items in an integrated virtual shop.

### ⚙️ Key Features
- **PWA (Progressive Web App):** Installable on iOS/Android with offline-ready caching strategies.
- **Fast QR Scanner:** Native camera integration (`html5-qrcode`) for millisecond student identification.
- **Virtual Shop:** Transactional system for students to buy rewards, automatically validating balances.
- **Role-Based Access Control (RBAC):** Admin and Teacher dashboards with granular permissions and passwordless login (Magic Link).
- **Automated Badge Generator:** Batch creation of ID cards using `html2canvas` for easy printing.

## 🇧🇷 Versão em Português

Um Progressive Web App (PWA) Full Stack desenvolvido para digitalizar o controle de frequência e criar uma economia de tokens focada na educação financeira de alunos.

### 🎯 O Problema
O registro manual de presença e a gestão de recompensas físicas geravam gargalos operacionais. Não havia um sistema centralizado para analisar o desempenho ou automatizar a entrega de incentivos.

### 💡 A Solução
Construção de uma aplicação web responsiva que atua como uma "fintech educacional". O sistema gerencia uma moeda virtual (Talentos), permitindo que os professores recompensem o engajamento. Os alunos utilizam um Crachá com QR Code para realizar check-ins e compras em uma loja virtual integrada.

### ⚙️ Funcionalidades Principais
- **PWA (Progressive Web App):** Instalável em iOS/Android com estratégias de cache para uso offline.
- **Scanner QR Rápido:** Integração nativa de câmera (`html5-qrcode`) para identificação de alunos em milissegundos.
- **Loja Virtual (Shop):** Sistema transacional onde alunos compram recompensas, com validação automática de saldo.
- **Controle de Acesso (RBAC):** Dashboards para Administradores e Professores com permissões granulares e login seguro sem senha (Magic Link).
- **Gerador Automático de Crachás:** Criação em lote de IDs usando `html2canvas` para fácil impressão.

## 🛠️ Tech Stack
- **Front-end:** React.js, Tailwind CSS, Chart.js, HTML5.
- **Back-end:** PHP 8, RESTful API.
- **Database:** SQLite (High performance for fast read/write operations).
- **Security:** Anti-Brute Force mechanism, secure token generation.
